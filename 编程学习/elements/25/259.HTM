Exception 类，可以直接继承，或者间接继承，例如:

class MyError(Exception):
        def __init__(self, value):
            self.value = value
        def __str__(self):
            return repr(self.value)
try:
        raise MyError(2*2)
    except MyError as e:
        print('My exception occurred, value:', e.value)
